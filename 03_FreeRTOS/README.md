## The simples FreeRTOS project with LED blinking

Project includes FreeRTOS operating system and performs LED blinking
from task.

## Features
 - Clock source: HSE
 - SYSCLK: 180 MHz
 - APB1: 45 MHz
 - APB2: 90 MHz
 - CMSIS only
 - FreeRTOS

## Versions
  - IAR EWARM: v8.20
  - CMSIS: 4.5.0
  - FreeRTOS v10.0.0
